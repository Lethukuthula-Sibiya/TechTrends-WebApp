# Tech Trends Management System

## Overview
An ASP.NET Core web application demonstrating the Repository Pattern for managing projects and clients with CRUD operations and Azure deployment.

## Features
- Project management
- Client management
- CRUD operations
- Repository Pattern
- Azure App Service deployment

## Tech Stack
- C#
- ASP.NET Core
- Entity Framework Core
- SQL Server
- Azure

## Learning Outcomes
- Repository Pattern
- Dependency Injection
- Entity Framework Core
- Cloud deployment
